# goofish_img

一个收集各大厂的图片接口的图床 使用全球加速的cdn 让您的图片快人一步加载！

# 配置教程

访问[闲鱼创作者平台](https://author.goofish.com/#/) 登录后按F12 复制你的cookie2 将它填入config.php文件中
